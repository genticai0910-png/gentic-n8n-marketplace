## For Browser: Upload Instructions

### Quick Upload Checklist

Each product includes:
- ✅ ZIP package (ready for upload)
- ✅ Gumroad listing template (`docs/GUMROAD_LISTING.md`)
- ✅ Setup documentation (`README.md`)
- ✅ Test payloads (horizontal) or templates (vertical)
- ✅ Suggested pricing already calculated

### Upload Process (Per Product)

**Step 1: Review Product**
```
Read [product-name]/docs/GUMROAD_LISTING.md
```

**Step 2: Download ZIP**
```
[product-name]-gumroad.zip
```

**Step 3: Go to Gumroad**
- Visit: https://gumroad.com/products/new
- Upload the ZIP file
- Copy/paste listing details
- Set suggested price
- Publish

**Repeat for all 20 products**
